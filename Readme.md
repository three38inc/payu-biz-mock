## NODE.JS PayUBiz Kit
----------------

### Files/Folders required:-
```
|-- payubiz_nodejs-master
    |-- .gitignore
    |-- Readme.md
    |-- checkout.html
    |-- package-lock.json
    |-- package.json
    |-- response.html
    |-- server.js
    |-- images
        |-- logo.png
```

### npm packages used:
- express
- express-session
- body-parser
- path
- crypto
- request
- ejs


### Getting Started
 ```
 npm install

 npm run start / node server.js / node server

 ```

Server URL and PORT :- http://localhost:3000

** Run server with 'node server' command. Open browser and type 'localhost:3000'.
